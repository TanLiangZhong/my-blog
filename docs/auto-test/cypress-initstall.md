### 安装node环境
> 官网下载node并安装.[下载地址](https://nodejs.org/zh-cn/)

### yarn 安装
```shell
npm install -g yarn
```

### cypress 安装
- 创建目录
```
mkdir learn-pypress
```
- 创建node项目
    ```shell
    npm init -y
    ```
- 安装 cypress
    ```shell
    yarn add cypress --dev
    ```
- 启动
    ```shell
    yarn cypress open
    ```
